NEXLAB 26.31.16 RC11 — R56.0.16

  • Resultados de filtros não carregados deixam de exibir dados do filtro anterior.
  • Cache isolado por usuário, impressão digital das permissões e parâmetros completos da consulta.
  • Atualização manual invalida todas as consultas do escopo atual.
  • A API ativa exige space_id para reservas e reuniões presenciais ou híbridas.
  • Espaços inexistentes ou inativos são rejeitados pelo backend.
  • Nome do espaço é mantido somente como snapshot para exibição.
  • Constraint GiST impede sobreposição de horários para o mesmo participante.
  • Alteração de horário ou status também revalida conflitos.
  • Matriz autenticada aprovada para Administrador, Coordenador, Bolsista e Coworking Júnior.
  • Correções cumulativas das RC1 a RC10 preservadas.
